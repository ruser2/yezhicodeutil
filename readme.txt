代码生成执行  code.CodeUtil下面的main方法。  默认生成   两个mapper.xml文件   xxMapper.java  xx.java  xxService.java  xxServiceImpl.java  xxAct.java文件
有不需要生成的class文件可以到  codeGenerate.factory.CodeGenerateFactory  注释掉对应生成的代码

resources.jeecg
	template	模板文件
		ControllerTemplate.ftl		act模板文件
		DaoTemplate.ftl			mapper模板文件
		EntityTemplate.ftl		domain模板文件
		MapperTemplate.xml		默认增删改成的xml模板文件
		MapperTemplate2.xml		默认为空的xml模板文件
		ServiceImplTemplate.ftl		service实现类的模板文件
		ServiceTemplate.ftl		service模板文件

		jeecg_config.properties		模板文件生成参数配置
			source_root_package	java文件放在根目录哪个包下面
			source_xml_package	xml文件放在根目录哪个包下面
			webroot_package		hmtl jsp js等文件生成放在根目录的哪个包下面
			

			templatepath		模板文件包路径
			system_encoding		编码
			
			bussi_package		生成的class通用的包路径 如：com.yllt
			project_path		生成文件放在哪个工程的目录下该路径为绝对路径
			
			jeecg_generate_table_id			未使用
			jeecg_ui_search_filed_num		未使用
			
			organization		代码注释上的organization信息

		jeecg_database.properties	
			diver_name		数据库驱动
			url			数据库连接
			username		用户名
			password		密码
			database_name		使用哪个数据库即数据库库名
